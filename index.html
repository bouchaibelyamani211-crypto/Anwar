Voici un code HTML/CSS/JS complet qui calcule le moment quadratique (moment d'inertie) pour différentes sections géométriques courantes, exécutable directement dans votre navigateur : ```html Calculateur de Moment Quadratique

📐 Calculateur de Moment Quadratique

Calcul du moment d'inertie pour les sections courantes

Sélectionnez un type de section et entrez les dimensions

📊 Résultats

Ix (mm⁴) : -
Iy (mm⁴) : -
Ix (cm⁴) : -
Iy (cm⁴) : -
Aire (mm²) : -
Moment polaire J₀ (mm⁴) : -
``` Ce calculateur offre les fonctionnalités suivantes : 📋 Fonctionnalités : 1. 8 types de sections : Rectangle, Cercle, Cercle creux, Carré, Carré creux, Triangle, Profilé H, Profilé I 2. Calculs précis : · Moment quadratique Ix (mm⁴ et cm⁴) · Moment quadratique Iy (mm⁴ et cm⁴) · Aire de la section (mm²) · Moment polaire J₀ pour les sections circulaires 3. Formules utilisées : · Rectangle : Ix = bh³/12, Iy = hb³/12 · Cercle : I = πD⁴/64 · Profilés : Méthode de décomposition avec théorème de Huygens 4. Validation des données avec messages d'erreur clairs 5. Interface intuitive avec sélection dynamique des paramètres 🚀 Pour l'utiliser : 1. Copiez le code dans un fichier .html 2. Ouvrez-le dans votre navigateur 3. Sélectionnez le type de section 4. Entrez les dimensions en mm 5. Cliquez sur "Calculer" ou appuyez sur Entrée Le design est responsive et fonctionne sur mobile comme sur desktop !
